## 🚀 Projects

### 🖊️ QuickDraw Sketch Classifier — NN vs. CNN
> Real-time sketch classification system comparing a fully connected Neural Network with a Convolutional Neural Network, trained on the Google QuickDraw dataset.

- Designed and trained both NN and CNN models from scratch using **PyTorch**
- Built a **real-time drawing interface** for live predictions with parallel inference (both models run simultaneously)
- CNN achieved **96.2% accuracy** vs. 93.4% for NN — with ~50% fewer parameters
- Includes training curves, confusion matrices, and failure analysis

**Stack:** `Python` `PyTorch` `CNN` `Deep Learning` `Computer Vision`  
🔗 [View Repository](https://github.com/alshamimo/quickdraw-sketch-classifier)
